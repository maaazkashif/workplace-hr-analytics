# IBM HR Analytics Dashboard

## Description

This repository contains a Power BI dashboard built using IBM Employee Attrition data to explore and answer key questions about employee turnover. The key insights derived from the dashboard are documented in the results file. This project serves as a starting point for anyone looking to learn how to use Power BI for data analysis and visualization.

## Project Learnings

* Developed an interactive Power BI dashboard to analyze employee attrition and performance trends.
* Applied advanced filtering, slicers, and drill-down features to enable dynamic data exploration.
* Created calculated columns and measures using DAX to transform and analyze data effectively.
* Designed a variety of visualizations including bar charts, clustered bar charts, line charts, area charts, matrices, slicers, and KPI cards.
* Derived actionable insights and recommendations to support strategies for improving employee retention.

## Conclusion

The project successfully addressed key questions related to employee performance and attrition patterns. The findings and insights from the analysis are documented in the results file, which can be accessed here: https://github.com/maaazkashif/workplace-hr-analytics/blob/main/results.txt
